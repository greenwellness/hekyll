_Waarom?_ GitHub is wereldleider op het gebied van broncode beheersystemen (specifiek dus Git). Andere soortgelijke systemen zijn "SVN":svn en "Mercurial":hg.
Geen is er zo krachtig en veelzijdig als GH(GitHub).

_Wat?_ Wij hebben een beperkt aantal zaken ondergebracht op GH:


|_.account     |_.repository                  |_.rollen                        |_.tech                          |
|              |                              |                                |                                |
|greenwellness | "greenwellness.github.io":io | tekst -> website generator     | "Jekyll":jrb (Ruby)            |   
|              | "documents":prose ^1^        | draft documentatie en notities | "Markdown":md, "Textile":ile   |
|              | hekyll                       | tekst -> presentatie generator | "Jekyll":jrb, "Impress.js":ijs |
|              |                              |                                |                                |

^1^ Authenticatie via OAuth2 koppeling naar GitHub
